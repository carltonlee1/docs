## Getting Started

The Pathfinder dashboard provides a visual interface for managing your routing configuration, viewing analytics, and testing prompts.

**Dashboard URL**: [https://pathfinder.naivana.io](https://pathfinder.naivana.io)

## Home Screen

When you log in, you'll see the main dashboard with key metrics:

[Screenshot placeholder: Dashboard home with metrics cards]

### Key Metrics Displayed

- **Total Routing Decisions**: Number of routing decisions made today/this week/this month
- **Average Latency**: Mean routing decision time
- **Cost Savings**: Estimated savings vs. always using premium models
- **Success Rate**: Percentage of successful completions

## Navigation

The sidebar provides access to all dashboard features:

| Section | Description |
|---------|-------------|
| **Home** | Dashboard overview and key metrics |
| **Playground** | Test prompts and compare routing decisions |
| **Analytics** | Detailed routing analytics and trends |
| **API Keys** | Create and manage API keys |
| **Providers** | Configure LLM provider credentials |
| **Settings** | Account and organization settings |

## Quick Actions

From the dashboard home, you can:

1. **Test a prompt**: Click "Try Playground" to test routing decisions
2. **Create API Key**: Generate a new key for integration
3. **View Recent Decisions**: See your latest routing decisions
