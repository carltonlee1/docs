## Model Selection Helper

Pathfinder supports two formats for model IDs:

### Native API Names (Direct Calls)
```python
"claude-sonnet-4-5-20250929"  # Anthropic native name
"gpt-5"                       # OpenAI native name
"gemini-2.5-pro"              # Google native name
```

### Helper Format (Convenience)
```python
"anthropic/claude-4.5-sonnet"  # Simplified format
"openai/gpt-5"                 # Simplified format
"google/gemini-2.5-flash"      # Simplified format
```

The helper format is automatically translated to native names via `get_llm_client_for_model()`.

## Model Capabilities

Each model supports different capabilities:

| Capability | Description | Supported Models |
|------------|-------------|------------------|
| `text_generation` | General text generation | All models |
| `code_generation` | Code writing and debugging | All models (especially Sonnet 4.5, GPT-5) |
| `reasoning` | Complex reasoning tasks | Opus 4.1, GPT-5, Gemini 2.5 Pro |
| `instruction_following` | Following complex instructions | All models |
| `long_context` | Processing long documents | All Claude 4.5 models (200K), Gemini 2.5 (1M) |
| `multimodal` | Image/video understanding | Gemini 2.5 Pro, GPT-5 (with vision) |

## Filtering by Capability

Request models with specific capabilities:

```python
decision = pathfinder.routes.decide(
    prompt="Write a Python function to analyze this dataset...",
    constraints={
        "required_capabilities": ["code_generation", "reasoning"]
    }
)
```

## Deprecated Models

**Do NOT use these outdated models:**

❌ **OpenRouter** - We call provider APIs directly
❌ **Claude 3.x models** (e.g., claude-3-opus-20240229) - Use Claude 4.5 series
❌ **GPT-4o models** - Use GPT-5 series
❌ **Gemini 2.0 models** - Use Gemini 2.5 series
❌ **Fictional model names** - Only real models from official providers

**Rationale**: Using real, currently available models ensures our code actually works. Fictional or outdated models cause 404 errors and confusion.

## Usage Examples

### Cost-Optimized Request

```python
# Let Pathfinder choose the cheapest model for simple tasks
decision = pathfinder.routes.decide(
    prompt="Translate 'hello' to Spanish",
    constraints={"max_cost_usd": 0.001}
)
# Result: Likely Gemini 2.5 Flash-Lite or GPT-5 Mini
```

### Quality-Optimized Request

```python
# Request best model for complex reasoning
decision = pathfinder.routes.decide(
    prompt="Explain quantum field theory and its implications...",
    constraints={
        "required_capabilities": ["reasoning", "long_context"],
        "max_cost_usd": 0.05
    }
)
# Result: Likely Claude Opus 4.1 or GPT-5
```

### Code Generation Request

```python
# Optimize for code generation
decision = pathfinder.routes.decide(
    prompt="Write a binary search tree implementation in Rust...",
    constraints={
        "required_capabilities": ["code_generation"],
        "max_cost_usd": 0.02
    }
)
# Result: Likely Claude Sonnet 4.5 (best for coding)
```
