## Complete Setup Guide

This guide walks you through getting all the credentials you need to use Pathfinder in your application.

## Step 1: Get Your Pathfinder API Key

### 1.1 Create Your Account

1. Go to [dashboard.pathfinder.ai](https://dashboard.pathfinder.ai)
2. Click **Sign Up**
3. Choose your authentication method (Email, GitHub, Google)
4. Complete the registration

### 1.2 Create an API Key

1. In the dashboard, click **API Keys** in the left sidebar
2. Click **Create New Key**
3. Give your key a name (e.g., "Production", "Development")
4. Click **Create**
5. **Important**: Copy the key immediately - you won't see it again!

```
Your key looks like: sk_live_a1b2c3d4e5f6g7h8...
```

### 1.3 Store Your Key Safely

Create a `.env` file in your project root:

```bash
# .env
PATHFINDER_API_KEY=sk_live_a1b2c3d4e5f6g7h8...
```

**Add to `.gitignore`:**

```bash
# .gitignore
.env
.env.local
.env.*.local
```

## Step 2: Set Up Model Provider Credentials

Pathfinder routes to OpenAI, Anthropic, Google, and other providers. You need API keys from the providers you plan to use.

### Option A: Just Getting Started?

If you're just testing, use **one provider** (we recommend OpenAI):

```bash
# .env
PATHFINDER_API_KEY=sk_live_xxx...
OPENAI_API_KEY=sk-...
```

### Option B: Full Setup (All Providers)

For production, get keys from all three major providers:

---

## Provider 1: OpenAI (GPT Models)

### Get Your OpenAI Key

1. Go to [platform.openai.com](https://platform.openai.com)
2. Click your profile → **API Keys**
3. Click **Create new secret key**
4. Copy the key

### Store It

```bash
# .env
OPENAI_API_KEY=sk-proj-xxxxxx...
```

### Test It

```python
from openai import OpenAI

client = OpenAI(api_key="sk-proj-...")
response = client.chat.completions.create(
    model="gpt-4-turbo-preview",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

---

## Provider 2: Anthropic (Claude Models)

### Get Your Anthropic Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Click **API Keys** in the left sidebar
3. Click **Create Key**
4. Copy the key

### Store It

```bash
# .env
ANTHROPIC_API_KEY=sk-ant-xxxxxx...
```

### Test It

```python
from anthropic import Anthropic

client = Anthropic(api_key="sk-ant-...")
response = client.messages.create(
    model="claude-3-opus-20240229",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.content[0].text)
```

---

## Provider 3: Google (Gemini Models)

### Get Your Google Key

1. Go to [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. Click **Create API Key**
3. Select your Google Cloud project (or create one)
4. Copy the key

### Store It

```bash
# .env
GOOGLE_API_KEY=AIzaSy...
```

### Test It

```python
import google.generativeai as genai

genai.configure(api_key="AIzaSy...")
model = genai.GenerativeModel('gemini-2.5-pro')
response = model.generate_content("Hello!")
print(response.text)
```

## Step 3: Verify All Credentials

Create a verification script to test all your credentials:

```python
# verify_credentials.py
import os
from dotenv import load_dotenv

load_dotenv()  # Load from .env file

print("Checking credentials...")

# Check Pathfinder
pf_key = os.getenv('PATHFINDER_API_KEY')
if pf_key and pf_key.startswith('sk_live_'):
    print("✅ Pathfinder API key configured")
else:
    print("❌ Pathfinder API key missing or invalid")

# Check OpenAI
openai_key = os.getenv('OPENAI_API_KEY')
if openai_key and openai_key.startswith('sk-'):
    print("✅ OpenAI API key configured")
else:
    print("⚠️  OpenAI API key not configured (optional)")

# Check Anthropic
anthropic_key = os.getenv('ANTHROPIC_API_KEY')
if anthropic_key and anthropic_key.startswith('sk-ant-'):
    print("✅ Anthropic API key configured")
else:
    print("⚠️  Anthropic API key not configured (optional)")

# Check Google
google_key = os.getenv('GOOGLE_API_KEY')
if google_key and google_key.startswith('AIza'):
    print("✅ Google API key configured")
else:
    print("⚠️  Google API key not configured (optional)")

print("\nAll required credentials are set up!")
```

Run it:

```bash
python verify_credentials.py
```

## Step 4: Use Credentials in Your Application

### Python Example

```python
# app.py
import os
from dotenv import load_dotenv
import requests
from openai import OpenAI

load_dotenv()

# Get credentials from environment
pathfinder_key = os.getenv('PATHFINDER_API_KEY')
openai_key = os.getenv('OPENAI_API_KEY')

# Step 1: Get routing decision from Pathfinder
response = requests.post(
    'https://api.pathfinder.ai/v1/routes:decide',
    headers={'Authorization': f'Bearer {pathfinder_key}'},
    json={'prompt': 'Write a hello world program'}
)

decision = response.json()
model = decision['selected_model']
provider = decision['provider']

print(f"Routing to {provider}/{model}")

# Step 2: Use the provider's API
if provider == 'openai':
    client = OpenAI(api_key=openai_key)
    completion = client.chat.completions.create(
        model=model,
        messages=[{'role': 'user', 'content': 'Write a hello world program'}]
    )
    print(completion.choices[0].message.content)
```

### Node.js Example

```javascript
// app.js
require('dotenv').config();

const axios = require('axios');
const { OpenAI } = require('openai');

const pathfinderKey = process.env.PATHFINDER_API_KEY;
const openaiKey = process.env.OPENAI_API_KEY;

async function main() {
  // Step 1: Get routing decision
  const response = await axios.post(
    'https://api.pathfinder.ai/v1/routes:decide',
    { prompt: 'Write a hello world program' },
    { headers: { Authorization: `Bearer ${pathfinderKey}` } }
  );

  const { selected_model, provider } = response.data;

  console.log(`Routing to ${provider}/${selected_model}`);

  // Step 2: Use the provider's API
  if (provider === 'openai') {
    const client = new OpenAI({ apiKey: openaiKey });
    const completion = await client.chat.completions.create({
      model: selected_model,
      messages: [{ role: 'user', content: 'Write a hello world program' }]
    });
    console.log(completion.choices[0].message.content);
  }
}

main();
```

## Step 5: Deploy to Production

### Environment Variables in Production

**Never commit `.env` files!** Use your platform's secrets management:

#### Vercel

```bash
vercel env add PATHFINDER_API_KEY
vercel env add OPENAI_API_KEY
```

#### AWS Lambda

```python
import json
import boto3

secrets_client = boto3.client('secretsmanager')

def get_secret(secret_name):
    response = secrets_client.get_secret_value(SecretId=secret_name)
    return json.loads(response['SecretString'])

# In your Lambda function
pathfinder_key = get_secret('pathfinder-api-key')
```

#### Google Cloud

```python
from google.cloud import secretmanager

def access_secret(project_id, secret_id, version_id='latest'):
    client = secretmanager.SecretManagerServiceClient()
    name = f"projects/{project_id}/secrets/{secret_id}/versions/{version_id}"
    response = client.access_secret_version(request={"name": name})
    return response.payload.data.decode('UTF-8')

pathfinder_key = access_secret('my-project', 'pathfinder-api-key')
```

#### Docker

```dockerfile
# Dockerfile
FROM python:3.11

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

# Secrets passed at runtime
ENV PATHFINDER_API_KEY=""
ENV OPENAI_API_KEY=""

CMD ["python", "app.py"]
```

Run with:

```bash
docker run \
  -e PATHFINDER_API_KEY=sk_live_xxx \
  -e OPENAI_API_KEY=sk-xxx \
  my-app
```

## Security Checklist

Before deploying to production:

- [ ] `.env` file is in `.gitignore`
- [ ] API keys are stored in environment variables
- [ ] No API keys hardcoded in source code
- [ ] Using production keys (`sk_live_`) only in production
- [ ] Using test keys (`sk_test_`) in development
- [ ] Secrets manager configured (AWS/GCP/Azure)
- [ ] API keys rotated at least monthly
- [ ] Old keys deleted from dashboard
- [ ] Team members have read-only access to secrets

## Troubleshooting

### "Module 'dotenv' not found"

Install python-dotenv:

```bash
pip install python-dotenv
```

### "API key not found in environment"

Check:
1. `.env` file exists in project root
2. Variable name matches exactly (case-sensitive)
3. `.env` file is not in `.gitignore` (it shouldn't be!)
4. You ran `load_dotenv()` before accessing environment variables

### "401 Unauthorized from Pathfinder"

Check:
1. Key starts with `sk_live_` (not `sk_test_`)
2. Key is copied completely with no extra spaces
3. Key is in `Authorization: Bearer` header (not `API-Key`)
