Pathfinder provides a simple REST API for intelligent LLM routing.

### Base URL

All API requests should be made to:

```
https://api.pathfinder.ai/v1
```

### Authentication

All endpoints require authentication via Bearer token:

```bash
Authorization: Bearer sk_live_your_api_key_here
```

See [Authentication Guide](/docs/api/getting-started/authentication) for details.

### Rate Limits

| Tier | Requests per minute |
|------|---------------------|
| Free | 60 |
| Pro | 600 |
| Enterprise | Custom |

### Common Headers

```http
Content-Type: application/json
Authorization: Bearer sk_live_...
X-Tenant-ID: your-tenant-id (optional, extracted from token)
```
